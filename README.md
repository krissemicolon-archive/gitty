# perplex
> Git Client in a TUI format

### Building
-> You need\
    - ghc\
    - cabal

```sh
git clone https://github.com/krissemicolon/perplex.git
cd perplex
cabal build
```

### Dependencies
- base >=4.13 && <4.14
- [brick](https://hmckage.haskell.org/package/brick)
- [gitlib](https://hackage.haskell.org/package/gitlib)
